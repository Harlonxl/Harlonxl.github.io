深度学习第三期

  • 课程大纲

    • 第一周 夯实DL必备基础
      第1课 夯实深度学习数据基础（管）
    1. 必要的微积分、概率统计基础
    2. 必要的矩阵、凸优化基础
    3. 实战：numpy与高效计算
      第2课 DNN与混合网络：google Wide&Deep（寒）
    4. 多分类softmax与交叉熵损失
    5. 人工神经网络与BP+SGD优化
    6. 实战：数据非线性切分+google wide&deep 模型实现分类

    • 第二周 从CNN入手，掌握主流DL框架
      第3课 CNN:从AlexNet到ResNet（寒）
    1. 卷积神经网络层级结构详解，可视化理解
    2. 典型卷积神经网络结构(AlexNet,VGG,GoogLeNet,ResNet)讲解
    3. 实战：搭建CNN完成图像分类示例
      第4课 NN框架：caffe, tensorflow与pytorch（寒）
    4. Caffe的便捷图像应用
    5. TensorFlow与搭积木一样方便的Keras
    6. facebook的新秀pytorch
    7. 实战：用几大框架完成DNN与CNN网络搭建与分类

    • 第三周 CNN延伸：生成对抗与图像风格转化
      第5课 造出你要的视界：生成对抗网络GAN（寒）
    1. 无监督学习与图像生成
    2. 生成对抗网络与原理
    3. 实战：DCGAN图像生成
      第6课 图像风格转化（加）
    4. 秒变文艺：neural style将照片转换成大师佳作
    5. 手推公式理解neural style原理
    6. 实战：neural-style与fast neural-style代码讲解

    • 第四周 掌握自然语言处理中的神经网络
      第7课 RNN/LSTM/Grid LSTM（寒）
    1. 序列数据与循环神经网络
    2. RNN/LSTM/Grid LSTM
    3. 实战：RNN文本分类
      第8课 RNN条件生成与attention（寒）
    4. RNN条件生成与attention
    5. “看图说话”原理
    6. google神经网络翻译系统

    • 第五周 迁移学习与增强学习
      第9课 增强学习与Deep Q Network（寒）
    1. 马尔科夫决策过程
    2. 价值函数与策略评价、学习
    3. Deep Q network
    4. 实战：用Tensorflow搭建Deep Q learning玩Flappy bird
      第10课 物体检测与迁移学习（寒）
    5. RCNN，Fast-RCNN到Faster-RCNN
    6. Fine-tune，保守训练，层转移，多任务学习
    7. 领域对抗训练
